# mobile-device-management

## Dependencies

java sdk, make, git

## Server

Handles the client requests and handles the database. Server must be running before the client.

Execute the following commands...

cd src/server

make clean

make

make run

## Client

UI client to send requests to the server.

Execute the following commands...

cd src/client

make clean

make

make run
